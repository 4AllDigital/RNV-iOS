# RNV-iOS
Demo RNV iOS issue.

```
"react": "16.9.0",
"react-native": "0.61.3",
"react-native-video": "^5.0.2"
```

### setup and installs dependencies
```
npm install
```

### setup and install native dependencies
```
cd ios && pod install
```

### start metor bundler
```
npm start
```

### open project in xcode
```
open ios/mobileVOD.xcworkspace
```
